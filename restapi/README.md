# このアプリについて
Go 言語のフレームワークである `Echo` の理解を深めるためにアレコレすることを目的としたものです。 したがって、完全に個人での利用を目的としたものであり、更新のたびに以前の機能が無くなることがあります。

学習の内容が気になる場合は各ブランチをご参照ください。
